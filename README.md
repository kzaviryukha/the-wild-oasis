# The Wild Oasis

A training project in which the following React's tech stack were used: React Router for routing, Styled Components for styling, React Query for remote state management, Context API for UI state management, React Hook Form for form management, React Icons, React Hot Toasts, Rechart, date-fns, Supabase.

This project was bootstrapped with Vite v.4.

## DEMO

You can visit the [DEMO](https://m9iv.github.io/kzaviryukha.github.io/demo/the-wild-oasis/index.html) page.
